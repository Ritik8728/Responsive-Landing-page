Responsive Landing Page

Overview

This project is a responsive landing page created using HTML, CSS, and JavaScript. It features an interactive navigation menu that remains visible across all pages and changes its appearance when users scroll or hover over menu items.

Features

- Fixed navigation bar
- Scroll-based style changes
- Interactive hover effects
- Responsive design
- Clean and modern UI

Technologies Used

- HTML5
- CSS3
- JavaScript

Project Objective

To develop a responsive and interactive landing page that provides a smooth and engaging user experience across different devices.
